全自動・顔モザイクメーカー (Face Mosaic App)

ブラウザ上で動作する、プライバシー保護に特化した顔モザイク加工Webアプリです。
AIが写真や動画内の顔を自動検出し、一括でモザイク処理を行います。

✨ 特徴

🔒 安心のプライバシー保護

アップロードされた画像・動画データはサーバーに送信されず、すべてお使いのデバイス（ブラウザ）内で処理されます（Client-side processing）。

🚀 超高精度AI搭載

face-api.js (SSD MobileNet V1 + Landmarks) を使用。

高精度の検出設定（信頼度閾値の調整済み）により、群衆、横顔、小さな顔なども逃さず検出します。

✍️ 手動補正モード

AIが見逃した箇所や、車のナンバープレートなど背景の映り込みを、マウスドラッグで直感的に隠すことができます。

🎥 動画対応 (Beta機能)

動画ファイル内の顔もフレームごとに追跡してモザイクをかける機能を試験的に搭載しています。

高画質（高ビットレート）での書き出しに対応。

📱 レスポンシブデザイン

PCだけでなく、スマートフォンやタブレットでも快適に動作します。

🛠️ 使い方

AIモデルの準備: ページを開くと自動的にAIモデルの読み込みが始まります。「AIモデル準備完了」になるまで数秒待ちます。

ファイルを選択:

画像（PNG, JPG）または動画（MP4, WEBM）をアップロードエリアにドラッグ＆ドロップ、またはクリックして選択します。

モザイク処理:

画像の場合: 「顔にモザイクをかける」ボタンをクリックします。

動画の場合: 「動画処理を開始」ボタンをクリックします（動画が再生されながらリアルタイムで処理・録画されます）。

手動補正 (画像のみ):

「手動でモザイク追加」をONにし、画像上の隠したい部分をドラッグして範囲選択します。

保存:

「保存」ボタンをクリックして、加工後のデータをダウンロードします。

💻 技術スタック

このアプリは外部バックエンドサーバーを必要とせず、フロントエンド技術のみで構築されています。

HTML5 / JavaScript: Vanilla JS (フレームワークなし)

CSS: Tailwind CSS (CDN)

AI/ML: face-api.js

TensorFlow.js 上で動作する顔認識ライブラリ

フォント: Google Fonts (Noto Sans JP)

🚀 インストール・実行方法

このアプリは単一の HTML ファイル (face_mosaic_app.html) で構成されています。

ローカルでの実行について

ブラウザのセキュリティ制限（CORSポリシー）により、HTMLファイルを直接ダブルクリックして開く（file:// プロトコル）と、AIモデルや画像の読み込みに失敗する場合があります。
必ずローカルサーバーを経由して実行してください。

方法 1: VS Code "Live Server" (推奨)

Visual Studio Code でフォルダを開く。

拡張機能「Live Server」をインストール。

HTMLファイルを右クリックし、「Open with Live Server」を選択。

方法 2: Python

# プロジェクトフォルダ内で実行
python -m http.server 8000


ブラウザで http://localhost:8000/face_mosaic_app.html にアクセス。

⚠️ 注意事項

動画機能: 動画処理はPCスペックに依存するため、長い動画や高解像度の動画では処理に時間がかかったり、動作が重くなる場合があります。

ブラウザ互換性: 最新の Chrome, Edge, Firefox, Safari での動作を推奨します。
