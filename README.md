🛡️ 全自動・高精度 AI顔モザイクメーカー
～ 面倒な「顔隠し」作業は、AIにおまかせ。 ～

写真をSNSにアップしたいけれど、通行人の顔や友人のプライバシーを守るためにスタンプを貼る作業が面倒…。そんな悩みを一発で解決するWebアプリです。

✨ 3つの大きな特徴
1. AIが全員の顔を自動検出 ボタンを1回押すだけで、AIが写真に写っている人の顔をすべて見つけ出し、自動でモザイク処理を行います。

2. 誤検知ゼロを目指した「超高精度モード」 単に肌色を認識するだけでなく、目・鼻・口の配置までAIが厳密にチェック。壁の模様や背景を「顔」と間違えてモザイクをかけてしまう失敗を極限まで減らしました。

3. 安心のプライバシー設計 このアプリは、読み込んだ写真を外部のサーバーに送信しません。すべての処理があなたのブラウザ（スマホやPC）の中で完結するため、誰にも見られずに安全に加工できます。

🛠️ こだわりの機能
手動補正モード: AIが見逃してしまった横顔や、背景の映り込み（車のナンバーなど）も、指やマウスでなぞるだけで追加でモザイクをかけられます。

PC・スマホ両対応: インストール不要。ブラウザを開けばすぐに使えます。
